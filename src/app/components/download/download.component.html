How to play:

Portable:

Press the relative download button and wait until process finishes, then open windows file explorer and navigate to your downloads folder (Usually its "C:\users\you\downloads" by default).

After that, right click on the file you just downloaded and select one of the extract options (depending of your computer, you may need to have installed a compressor such as WinRar).

Lasty, open the new extracted folder and double click the .exe file that has the same name as the game and the program will start.

Installer:

Press the relative download button and wait until process finishes, then open windows file explorer and navigate to your downloads folder (Usually its "C:\users\you\downloads" by default).

After that, double click on the file you just downloaded and a window will pop-up (You may first be prompted to run the program as administrator), so just follow the install wizard instrucions.
